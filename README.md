# Crypto Live Price Tracker 
This is a Python tool that tracks real-time prices of the top 10 cryptocurrencies and saves them into a CSV file.

# Features
- Scrapes live data from API.
- Saves data to crypto_live_data.csv on the Desktop.
- Updates every 5 minutes.

# Requirements
- Python 3.x
- pandas, requests libraries

# How to Run
1. Install libraries: pip install requests pandas
2. Run the script: python crypto_live_tracker.py
